Listado de Combos

1. Golpes Fuertes

Input: ■ + ■ + ■ (o ■ + ■ + ■ + ■, dependiendo del personaje) + ▲ + ▲ + ▲ + ■

2. Volar y Atacar desde Atrás

Input: Mantener aplastado R2 + ✘ + ✘ + ▲ + ■ + ▲ (mientras mantienes R2)

3. Super Combo

Input: Energía al máximo + ■ ↑ + ✘ + ■ + ✘ + ■ + ✘ + ■ + ▲↑ + ▲↓ + ▲← + ▲→

4. Mientras el enemigo está atacando

Input: Cuando el enemigo está atacando con 5 cuadrados (■ + ■ + ■ + ■ + ■), cubre con 𒊹 y luego contraataca con ▲.

5. Hacer Contra Z

Input: Justo después de recibir un ataque, usa ↑■, luego presiona 𒊹.

6. Combo Infinito

Input: ■ + ▲ + ■ ➞ 𒊹
