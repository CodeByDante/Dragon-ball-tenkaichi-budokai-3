4. Combo de Esquivar y Contraataque

Descripción: Cuando el enemigo está atacando con 5 veces cuadrado (■ + ■ + ■ + ■ + ■) y tú te cubres con 𒊹, puedes contraatacar presionando ▲.

Input: ■ + ■ + ■ + ■ + ■ (enemigo) → 𒊹 (cubrir) → ▲ (contraataque)

Método 2: ✘↑ + ↑▲ + ⬈R1 + ⬈R2

6. Hacer Contra Z

Input: Justo después de recibir un ataque, usa ↑■, luego presiona 𒊹.

7. Combo Infinito

Input: ■ + ▲ + ■ ← 𒊹

8. Delisada (para buscar al enemigo por la espalda)

Input: ▲↑ + R1 + ✘

Uso: Este combo es útil para buscar al enemigo por la espalda, permitiendo un ataque sorpresa.

9. Combo Complementario a Delisada

Input: ■ + ■ + ■ + ■ + ■ + ✘ ↑ + ▲ + 5 ↑ + R1 + ✘

Uso: Este combo complementa el "Delisada" para aumentar la eficacia del ataque.
